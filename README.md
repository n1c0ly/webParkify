Parkify — Sistema de Controle de Vagas

Um sistema web para gerenciamento de estacionamento 24 horas, desenvolvido em React, permitindo visualizar vagas, registrar entrada de veículos, retirar veículos e validar placas automaticamente.
Criado para ser simples, rápido e eficiente para administradores.

Funcionalidades

 Listagem das vagas (20 vagas configuradas)

 Registrar entrada com validação de placa

 Marcar vaga como ocupada ou liberada

 Confirmação antes de retirar o veículo

 Interface estilizada com CSS personalizado

 Sistema totalmente frontend, sem dependência de API externa

 Design simples

Tecnologias Utilizadas

React + Vite

JavaScript

CSS puro

React Hooks (useState, useEffect)
